    Commands

    • mkdocs new [dir-name] - Create a new project.
    • mkdocs serve - Start the live-reloading docs server.
    • mkdocs build - Build the documentation site.
    • mkdocs -h - Print help message and exit.

    Admonitions (Hervorgehobene Blöcke)

    Dokumentation

    Mit dem folgenden Code können in den .md-Dateien hervorgehobene Blöcke erstellt werden:

    !!! note
    Lorem ipsum dolor sit amet, consectetur adipiscing elit. Nulla et euismoda
    nulla. Curabitur feugiat, tortor non consequat finibus, justo purus auctor
    massa, nec semper lorem quam in massa.
    ??? success "Lösung"
        Hier steht eine Lösung

        über mehrere Zeilen.

    Aufgaben erstellen:

    Eine Aufgabe, kann in einer yaml-Datei gespeichert werden und dann wie folgt geladen werden:

    question: |
  Wie heißt
  du?
solution: >
  Ok, das ist 
  fein so
tip: Hier ist ein nützlicher Tipp
title: Erste Frage
difficulty: 3

    {{ task(file="ex1.yaml") }}

    Ausgabe:

    🌶🌶🌶 Erste Frage

    Wie heißt

    du?

    Tipp

    Hier ist ein nützlicher Tipp

    Lösung

    Ok, das ist fein so

    Aufgabenteile

    Eine Aufgabe besteht aus den folgenden Teilen:

    Parameter Default
    title Aufgabe
    question ⚠QUESTION_TEXT_MISSING⚠
    solution
    tip
    difficulty 0
    difficulty_icon '🌶'

    Aufgabe inline definieren

    Es kanna auch eine Aufgabe erstellt werden, indem die einzelnen Parameter direkt gesetzt werden. Es kann auch eine Datei als Grundlage genutzt werden. Übergebene Parameter überschreiben jedoch die Inhalte der Datei:

    {{ task(file="ex1.yaml", title="bessere Frage", 
solution="""
Los gehts

Hier ist die Antwort
""") }}
    🌶🌶🌶 bessere Frage

    Wie heißt

    du?

    Tipp

    Lösung
